Ratings

Ratings are used mainly to rate Products in E-Commerce Websites.

E-Commerce Ratings

Use class star__ratingto form a E-commerce rating We use 10 inputs one for half star.

Static Rating

Use class static__ratingto use a static rating you can use class rating__icon on icon to make it golden or can override it with color of your choice
