Avatars

The Avatar component is used to represent a user, and displays the profile picture, initials.

Round Avatars

Round avatar are basically used as Profile Pictures. You can use class avatar to use them

Sizes in Avatars

The Avatar component comes in 5 sizes. you can use class xs, s, m, l, xl to have different sizes of avatars

Square Avatars

Use class avatar-square to have square avatar you can also used different Sizes with it
