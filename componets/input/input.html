Input

Inputs are used to get the user information and to store that data. Inputs can be used in forms during signin and in many other places.

Basic Input

Use class input to form a basic input

Input with Different Size

use class input-s, input-m , input-l for Different Sizes

Labeled Inputs

Inputs with Label of your choice use class input__box and class input inside input for labeled inputs

Error Success Inputs

Primary , Success , And Error Inputs. Use class input__primary , input__error,input__success to use this inputs

Input With Messsage

Use input__message class on your Message to show success , Errro Message to user
