Images

Square Images

Square Images are basically in square shaped use class image__sqr to add square images in your webiste with a default dimension of 200px X 200px.

Round Images

Use class "image__round" for round images with a default dimension of 200px X 200px.

Images of Different Sizes

Use class image__s image__m image__l with image__round or image__sqr for different sizes.

Responsive Image

Responsive Images change width according to browser width. use class "image__res" for responsive Images

Change the browser window, to see the responsive property.
