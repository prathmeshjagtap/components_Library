Buttons

The Button component allows user to take action such as submitting a form, opening a dialog, canceling an action, or performing a delete operation.

Simple Buttons

Just add btn class to create a button . You can use additional classes like btn-primary, btn-warning, btn-success, btn-dark on your choice.

Outline Buttons

Add outline class such as btn-outline-primary , btn-outline-warning ,btn-outline-success ,btn-outline-dark to have different types of outline buttons

Icon Buttons

Buttons with different icons add class btn-icon to button and class icon to your icons . you can add additonal classes to buttons for different colors. You can also add svgs but just give class name icon.

Link Buttons

Link Buttons are used to link to other parts of page you can use this using btn and btn-link class
